1、下载VMWare安装包

比如：

VMware-Workstation-Full-10.0.1-1379776.x86_64.bundle

【注意选择64位还是32位】


2、在网上找到注册序列号，例如

任意版本永久KEY注册密钥：

CZ7E2-2FY5P-08DUP-6XNXG-WPKE8

ZC7WU-2VZ95-08EMZ-R5P5X-MGU94

YF3XU-F3F01-4888Q-WENXC-NKUC8

FF34U-8FF9N-M81KY-A4ZGG-WAUZ8

ZC79R-F3Z15-H8E5Z-TXMEE-PUU8F

VF7WK-60E4L-M80JQ-2NQQZ-QKKED

YU7HA-03DD7-M84TY-WXWEE-ZQ2Z0

FC54R-FQF44-4812P-45XQV-PPACF


3、给下载来的 .bundle文件添加可执行权限

sudo chmod +x *.bundle


4、启动 .bundle安装程序

sudo ./*.bundle

【3、4可以合并为一步： sudo chmod +x *.bundle && sudo ./*.bundle 】


5、安装完成后，在软件管理器中搜索到VMware-Workstation打开即可使用

 【 Windows下安装的系统，在linux下仍然可以使用，例如我装在Windows D盘的XP：

/media/zollty/本地磁盘/C/vmware/WindowsXP/Windows XP Professional.vmx

在linux下的VMware-Workstation中仍然可以正常启动，建议安装在移动硬盘上，方便使用】
