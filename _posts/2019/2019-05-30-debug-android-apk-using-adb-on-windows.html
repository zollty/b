1、找到adb安装路径，如果安装了Android SDK，则在platform-tools下面，例如：

D:\devsoft\Android\SDK\platform-tools


2、运行 adb.exe --help查看命令

adb.exe devices -l 查看连接的设备（首次使用会启动adb服务，连上android数据线会提示是否允许调试，如果在手机端同意调试，则adb会自动连接上这个设备）


3、adb.exe install xxx.apk 安装apk（也可以手动自己安装）


4、adb logcat 根据TAG过滤

adb.exe logcat -s Jetty:V

logcat的用法详见：adb.exe logcat --help
