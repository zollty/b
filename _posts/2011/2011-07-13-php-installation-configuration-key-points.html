首先，分为以下几种：

WIMP: Windows+IIS+MySQL+PHP

WAMP: Windows+Apache+MySQL+PHP

LAMP: Linux+Apache+MySQL+PHP

APPSERV：集成环境


    企业一般用Linux作为网站的服务器，所以一般都是LAMP架构；但是我们个人学习和开发，通常用Windows系统。

    推荐个人学习用APPSERV或者WAMP，不要用IIS，IIS安装和设置非常麻烦而且功能远远没有Apache专业。


APPSERV安装很简单，一键安装，安装好之后无需配置就可以直接使用。下面讲解WAMP安装的一些知识要点：

1. Apache的配置文件httpd.conf，需要添加下面3句（添加在#LoadModule vhost_alias_module这些行的后面）：

LoadModule php5_module C:/php/php5.3.1/php5apache2_2.dll

PHPlniDir "C:/php/php5.3.1"

AddType application/x-httpd-php .php .phtml

2.PHP配置文件php.ini

php.ini-development、php.ini-production 修改成php.ini即可

extension_dir = "./"，设置为extension_dir = "C:/php/php5.3.1/ext"

手动开启组件：

;extension=php_bz2.dll

;extension=php_gd2.dll

;extension=php_mysql.dll

;extension=php_mysqli.dll

等等，只需把前面的分号去掉即可。

注意：PHP可能与不同版本的Apache搭配效果不一样，测试版本为：

Apache 2.2.14 安装版

PHP 5.3.1 VC6 x86 Thread Safe（下载地址：http://windows.php.net/download/）

Mysql 5.0.x 安装版或者解压版都可以

测试PHP能否连接到Mysql：

<?php
 $ok=mysql_connect("localhost","root","");
 if($ok){
  echo "连接成功";
 }
?>
