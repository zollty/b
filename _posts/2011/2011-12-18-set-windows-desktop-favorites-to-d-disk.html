PS：我们有很多东西放在桌面上，但是一旦系统出了故障要还原或者重装，桌面上的东西就没救了（当然，如果你是高手，当系统出了故障进不去时，可以试着用U盘启动，然后把C盘下桌面上的东西拷贝出来）。 

最好的办法是，将桌面目录设置到其他盘（比如D盘）。 C盘中，尽量不要放重要的资料。

 

我以前系统坏过一次，桌面上好多重要的资料都丢了。

 

方法如下：

 

1、打开注册表：运行（WIN键+R键），输入regedit

2、依次点开：

HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\Shell Folders

找到右面的Desktop项，双击修改其路径，我在E盘新建了E:\Users\Desktop，然后把这个路径粘贴进去。

 

3、再找到：

…(同上)\Explorer\User Shell Folders

找到右面的Desktop项，双击修改成和上面一样的路径。

 

注销或重启后就可以用了，然后把原来桌面上的东西拷贝到新的桌面上。


同理，设置收藏夹、IE收藏夹也是在上面那个路径下：

…(同上)\Explorer\Shell Folders

…(同上)\Explorer\User Shell Folders

找到"Favorites"这一项，修改其值即可。
